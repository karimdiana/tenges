## üìê Image Specifications
- **Format:** JPG, PNG, or WebP
- **Size:** 500x500px (square format)
- **Quality:** High resolution for best display
- **File naming:** Exactly as shown above (case-sensitive)

## üîÑ How to Add Images
1. Create folder `public/images/` if it doesn't exist
2. Add your images with the exact file names listed above
3. Images will automatically appear on the website
4. No code changes needed once images are in place
